# fazo-client-cs

Simplistic screenshot uploading tool

# Configuration File

Before use, create `conf.json` file in the same directory as the `fazo-client-cs.exe`

```json
{
  "host": "http://localhost:9000",
  "shortcut": {
    "key": "E",
    "modifiers": [
      "Control",
      "Shift"
    ]
  }
}
```
